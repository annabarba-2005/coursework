# CSS Architecture - Nested BEM Structure

## Описание

Проект использует методологию **Nested BEM** для организации CSS-стилей. Это обеспечивает модульность, переиспользуемость и простоту поддержки кода.

## Структура файлов

```
css/
├── main.css                    # Главный файл с импортами
├── base/                       # Базовые стили
│   ├── _variables.css          # CSS-переменные
│   ├── _reset.css              # Сброс браузерных стилей
│   └── _typography.css         # Типографика
├── blocks/                     # БЭМ-блоки (компоненты)
│   ├── header/
│   │   └── header.css          # Шапка сайта
│   ├── hero/
│   │   └── hero.css            # Hero-секция
│   ├── product-card/
│   │   └── product-card.css    # Карточка товара
│   └── ...                     # Остальные блоки
└── utilities/
    └── _utilities.css          # Утилитарные классы
```

## Правила именования БЭМ

### Блок (Block)
Независимый компонент страницы.
```css
.header { }
.product-card { }
.btn { }
```

### Элемент (Element)
Часть блока, которая имеет смысл только внутри блока.
```css
.header__logo { }
.product-card__title { }
.btn__icon { }
```

### Модификатор (Modifier)
Флаг, который меняет внешний вид или поведение блока/элемента.
```css
.btn--primary { }
.btn--large { }
.product-card--featured { }
```

## Использование

Все стили подключаются через один файл `main.css`:

```html
<link rel="stylesheet" href="css/main.css">
```

## Адаптивность

Медиа-запросы следуют breakpoints из переменных:
- **Desktop**: от 1280px
- **Tablet**: от 768px  
- **Mobile**: от 320px

```css
@media (max-width: 992px) { }
@media (max-width: 768px) { }
@media (max-width: 576px) { }
```

## CSS-переменные

Все цвета, отступы, шрифты и прочие значения определены в `base/_variables.css`.

```css
:root {
    --color-primary: #2E7D32;
    --spacing-md: 1rem;
    --font-heading: 'Montserrat', sans-serif;
}
```
